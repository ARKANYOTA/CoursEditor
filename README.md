### Dépendances

- gcc (pour compiler) ou autre compilateur c++

L'installer sous arch:
```bash
yay -Sy gcc
```

### Compiler et exécuter
#### Compiler le code

```bash
make compile
ou
g++ main.cpp
```

#### Exécuter le code

```bash
make execute
ou
./a.out
```

#### Faire les 2 en meme temps

```bash
make a
```
